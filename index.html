Welcome to AQ

AQ is a lightweight, robust audio engine for HTML5. It was created in 2013 by Brian Moriarty as a component of the Perlenspiel 3 game engine.

The AQ engine is especially good at detecting and reporting errors. If an audio call fails for some reason, AQ can usually help you figure out why. You can easily configure AQ to display error and debugging strings through your own application's messaging system.

AQ resides in a single .js file. It is entirely self-contained, and doesn't require JQuery or any other external libraries.

AQ uses the high-performance Web Audio API on browsers that support it, falling back gracefully to standard HTML5 audio on those that don't. Antique browsers that do not fully support HTML5 cannot be used with AQ.

Grateful acknowledgement is made to James Simpson, whose howler.js project served as an important resource and inspiration for AQ.
