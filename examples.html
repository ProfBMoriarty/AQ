Examples

This page provides examples of how to use the AQ engine.

Load and immediately play a sound

This code initializes the AQ engine, loads a sound named "quack" and plays it immediately. The file buffer is saved for later reference.

The call to AQ.init() specifies that the "quack" sound is located in a sub-directory called audio/ located in the same directory as the aq.js script, and that the sound is available in .ogg, .mp3 and .wav formats. A simple alert handler is provided.

The AQ.load() call loads the sound. The autoplay option in the parameter object tells the engine to play the sound immediately after loading. The return value, buffer, will contain a string uniquely identifying the file buffer associated with the "quack" sound. This string can be used in any subsequent calls to AQ.play().

var result, buffer;

result = AQ.init(
{
    defaultPath: "audio/",
    defaultFileTypes: [ "ogg", "mp3", "wav" ],
    onAlert: function (str) {
       window.alert(str);
    }
});

buffer = AQ.load("quack", { autoplay: true } );

Note that AQ.load() only returns a buffer id, not a channel id. This means you can't control a sound played using AQ.load() using any of the AQ functions that expect a channel id, such as AQ.pause() and AQ.stop(). If you need this functionality, call AS.load() without the autoplay option, and use the returned buffer id to call AQ.play(), which does return a channel id.
