Links

This page offers links of interest to users of the AQ engine.

Web Audio API

The Web Audio API is now the official standard for HTML5 audio. It's already working in Chrome and Safari, and should soon be supported by all major browsers.

WC3's official specification is here.

HTML5 Rocks has a good introduction.

howler.js

James Simpson's howler.js is a minimal HTML5 audio engine similar to AQ. It currently offers more functionality, but minimal error detection/reporting.
